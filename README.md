# ESP32 Wroom 32D Writer

Aquest projecte permet enviar missatges per Wi-Fi perquè es mostrin per una matriu LED que funciona amb el protocol ESP32.

## Estructura del projecte

```
ESP32-Wroom-32D-Writer/
  └── src/
    ├── Message.py
    └── send_message.py
```

## Ús

Per a fer servir aquest programa, edita les variables a `/src/main.py` per a que s'adjustin a les teves necessitats. Finalment, executa `main.py`.
